# Urban-healthy-lifestyle-analysis


THE DATASET:

Ranking of 44 cities worldwide (2021) from most to least healthy, based on different variables related to physical and mental health, living cost and quality, and environmental factors (source: Kaggle).


THE AIM:

Identify homogeneous lifestyle groups and determine key variables influencing urban well-being.


METHODOLOGIES USED:

Linear Regression

Cluster Analysis

Principal Component Analysis (PCA)
